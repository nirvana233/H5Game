# ⏱️ 一瞬 · Blink Rush

一个纯前端单文件的**子弹时间高速穿缝跑酷**，主打**高速 vs 慢镜的强烈张力 + 秒级刷分裂变**。全新原创玩法。

## 玩法
- 你快到看不清，正面全是必死的窄缝障碍
- **按住屏幕** → 时间近乎静止（子弹时间），同时**滑动手指**把飞梭对准缝隙
- **松手** → 世界瞬间提速爆冲，呼啸穿过
- 慢放会消耗**能量**（顶部蓝条），常速时回充；能量耗尽就慢不下来了
- **越快通过分越多**：极速穿缝、贴边险过都有额外加成与连击

## 核心爽点
- "时之主"般在高速死亡中冷静穿针的掌控快感
- 慢放=冷色暗角凝滞，极速=暖色速度线爆冲，节奏对比极强

## 运行
直接用浏览器打开 `index.html` 即可，零依赖、零构建。

```bash
open index.html        # macOS
```

## 技术要点
- 纯 `HTML/CSS/Canvas/JS` 单文件，无任何外部依赖
- 双时间轴：世界用 `timeScale` 缩放，控制/能量用真实时间 → 慢放手感顺滑、提速按真实时长累计
- 能量经济：慢放消耗 / 常速回充 / 耗尽禁用慢放，制造取舍
- 极速通过 + 贴边险过的风格加成与连击系统
- 冷暖背景插值、速度线、暗角蒙版、死亡闪白、屏幕抖动、WebAudio 音效
