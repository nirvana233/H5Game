# 🌙 一念之间（一句话神明）

一个纯文字、单文件的**叙事涌现神明经营**游戏。你是一方天意，每年只能向子民**降下一句话**，世界会照它自己的方式去领会——主打**短局重开 + 黑色幽默 + 多结局收集**。

## 玩法
- 每年遇一桩事，从几句「旨意」中**择其一**降下
- 你的话牵动四维国运：**👥 黎民 · 🔥 信仰 · 🌾 仓廪 · ☯️ 太平**
- 指标越界触发**危机连锁**（狂信 / 饥荒 / 暴乱 / 人满为患）
- 任一根基崩塌即终局，**信仰过盛亦成灾**；撑过三十年方见盛世
- **8 种结局**按活法解锁：太平盛世 · 隐神 · 狂信 · 荒村 · 燔燎……
- 结局图鉴自动存档（`localStorage`），晒出你的天命

## 运行
直接用浏览器打开 `index.html`，零依赖、零构建。

```bash
open index.html        # macOS
```

## 技术要点
- 纯 `HTML/CSS/JS` 单文件，深色水墨夜空 UI，移动端优先
- 数据驱动事件引擎：**16 个普通事件 + 4 个条件危机事件**，危机优先注入
- 后果支持固定 `fx` 与 `roll()` 随机掷骰（战争胜负、沉默被曲解等），保证每局不同
- 指标 0–100 钳制 + 浮动数字反馈；结局由四维边界 + 享国年数 + 历史最高信仰共同判定
- `localStorage` 持久化结局图鉴；WebAudio 音效可静音
