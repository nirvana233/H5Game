# 色相位移 · Hue Shift

> 你就是一种颜色——只有**和你同色**的平台才托得住你。半空切换颜色，整个世界在你脚下瞬间重组。

一款原创 2D 平台冒险 H5 小游戏。核心只有一招：**变色**。颜色不是装饰，而是「什么对你是实心的」。

## 核心玩法

- 🟪 **粉色平台**：你是粉色时实心，蓝色时变虚影、会穿过去
- 🟦 **蓝色平台**：反之
- ⬜ **米黄平台**：永远实心，安全立足点
- 最爽的操作：**起跳 → 半空变色 → 精准落在刚刚出现的平台上**
- ⚠️ 灰刺永远扎人；掉出画面会重来；★ 收集星星冲高分

四种体验融为一招：精准（压线切色）、探索（异色平台预览成虚影提示路线）、动量（红蓝节奏连切进入心流）、机制惊喜（同一空间换色即换路）。

## 操作

| | 移动 | 跳 | 变色 |
|---|---|---|---|
| **PC** | ← → / A D | 空格 / ↑ / W | Shift / J / K |
| **手机** | 左下方向键 | 右下「⤒」 | 右下「变色」 |

## 关卡

5 关递进：教学 → 深坑半空变色 → 高低交错 → 连续切色长廊（心流段）→ 综合终关。带最佳用时/星星本地记录与成绩分享。

## 技术

- 纯单文件 `index.html`，零依赖，Canvas 2D 渲染
- 固定步长物理循环、土狼时间 + 跳跃缓冲、短按矮跳
- WebAudio 程序化音效、粒子 / 震屏 / 闪光 juice
- 触屏 + 键盘双控，自适应缩放
- 所有关卡经跳跃物理仿真校验：相邻落脚面均可纯跳跃到达

直接用浏览器打开 `index.html` 即玩。
