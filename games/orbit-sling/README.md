# 🪐 引力弹弓 · Orbit Sling

> 借引力绕场命中的太空弹射解谜 · 单文件零依赖 H5 游戏
> 直接用浏览器打开 `index.html` 即可游玩（手机 / 桌面通吃）。

---

## 一句话

愤怒的小鸟靠直觉抛物线，**引力弹弓靠"读懂引力场"**——
太空里没有重力，只有星球、黑洞、脉冲星把你的弹道**拉弯**。
拖动瞄准时有**实时引力预测线**，再拖动你自己的**引力井**布置轨迹，
打出那条"绕一圈再命中"的神之一射。

## 核心玩法

```
拖动瞄准 → 看预测线被天体拉弯 → 拖动引力井微调轨迹 → 预测线变绿(命中) → 松手发射
              ↑ 本作灵魂：发射前的轨迹规划
```

- **实时引力预测线**：瞄准时实时积分弹道并画出，让玩家"看得见"引力如何弯曲轨迹——这是把"引力解谜"做得不劝退的关键。
- **你的引力井**（紫色发光点）：每关给 0~2 个，可任意拖放，亲手把弹道掰成你想要的形状。预测线随拖动实时更新。
- 预测线**命中目标时变绿并提示"命中航线 ✓"**，给出明确的"啊哈"反馈。

## 天体机关

| 天体 | 行为 |
|---|---|
| 🌍 **行星** | 正引力吸引弹道；撞上核心 = 这一发报废 |
| ⚫ **黑洞** | 超强引力，擦边可获得"引力弹弓"加速；被吞 = 报废 |
| 🔵 **脉冲星** | 斥力场把弹丸推开，借它转向 |
| 🌀 **虫洞** | 成对出现，进一个从另一个出来，速度方向不变 |
| ⭐ **星星** | 目标，命中即收集；一发连锁命中有分数倍率 |
| 🟣 **引力井** | 你的工具，拖放布置专属弹道 |

## 目标与计分

- 收集关卡内全部星星即净化星域；每关弹丸有限。
- **连锁**：同一发连续命中多颗星星，分数倍率叠加（神之一射的终极目标）。
- **剩弹奖励**：省下的弹丸每个 +900。
- 三星评定 + 本地最佳分（localStorage 自动存档）。

## 设计概念（Game Concept）

### 核心身份

| 项 | 内容 |
|---|---|
| 核心动词 | 弹射 + 借引力绕行（**发射前规划轨迹**） |
| 核心幻想 | "我算准了整个引力场"——台球 + 天体物理的智力优越感 |
| 独特钩子 | 像愤怒小鸟太空版，**而且**引力井可由玩家摆放，每关是一道可解的轨迹谜题 |
| 主导 MDA | Challenge（精通） + Discovery（轨迹"啊哈"时刻） |
| 规模 | 小到中（单文件，已含 12 关 MVP） |

### 设计支柱（Pillars）

1. **看得懂的引力** —— 任何引力效果都必须能被预测线清晰呈现，绝不让玩家"一脸懵"。
   - *设计测试*：若某机制无法在预测线上体现，要么可视化它，要么砍掉。
2. **规划即乐趣** —— 乐趣在发射前的"读图与调试"，发射后是验证。
   - *设计测试*：若一关只能靠运气乱试而非规划，重设天体布局。
3. **绕一圈的高光** —— 鼓励"绕场命中""一发连锁"，制造可截图分享的神之一射。
   - *设计测试*：若最优解总是直线直击，加入迫使绕行的天体。

### 反支柱（Anti-Pillars）

- **不做重力** —— 太空里只有天体引力，弹道完全由引力场决定，保证确定性与可预测。
- **不做隐藏信息** —— 不靠"看不见的陷阱"制造难度；难度来自引力场的复杂度。
- **不做体力 / 内购墙** —— 即点即玩，不卡进度。

### 视觉锚点（Visual Identity Anchor）

> **视觉铁律：引力是看得见的——一切弯曲都要被光线"画"出来。**

- 深空霓虹美学：靛蓝星空、发光天体、黑洞吸积盘、虫洞旋涡。
- 预测线是画面主角：点阵航线随引力弯曲，命中时整条线发绿脉冲。
- 高对比让"目标 / 引力源 / 危险体"一眼可辨。

### 目标玩家

- **主要**：解谜型玩家（Explorer / 喜欢"读懂系统、找到最优解"）。
- **次要**：技巧型玩家（追求一发连锁、最少引力井的极限分）。
- **不适合**：只想要即时动作爽感、不愿思考的玩家。

## 技术说明

- 纯 `Canvas 2D` + 原生 JS，**单文件、零依赖、零素材**（音效 WebAudio 程序化合成）。
- 牛顿引力数值积分（半隐式欧拉）；**预测线与实飞共用同一物理步进**（固定步长 `FIXED`），保证预测精确不骗人。
- 世界 1280×800 等比 contain 缩放，全局可见，便于规划。
- 存档：`localStorage`（星级 + 最佳分）。
- 真机手感调参集中在脚本顶部「调参」区（引力强度 `BODY_G` / `ANCHOR_G`、`MAX_SPEED`、`FIXED`、`PRED_STEPS` 等）。

## 与「弹弹变变」的关系

二者同源于「愤怒的小鸟」弹射底座，却是**互补的两种体验**：

- **弹弹变变 Sling Morph** = 发射后操控（动作 / 手法 / 子弹时间变身）
- **引力弹弓 Orbit Sling** = 发射前规划（智力 / 解谜 / 引力读图）
