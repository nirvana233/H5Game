# 💥 连锁一击 · Chain Bang

> 一次点火引爆鲁布·戈德堡连锁 · 单文件零依赖 H5 游戏
> 直接用浏览器打开 `index.html` 即可游玩（手机 / 桌面通吃）。

---

## 一句话

愤怒的小鸟一发砸一处，**连锁一击一发引爆全场**——
每关是一台**断了几环的鲁布·戈德堡机器**。
你拖放自己的炸药补全链路、瞄准点火，然后**看着自己算准的连锁哗啦炸开**，
一次点火引爆所有气球。

## 核心玩法

```
观察断裂的连锁机器 → 拖放你的炸药补全链路 → 瞄准点火(仅一次) → 观赏连锁清空全场
                          ↑ 本作灵魂：布局家的运筹 + 连锁演出
```

- **一次点火**：每关只有一发火花弹。命中第一个元件即触发连锁。
- **补全链路**：关卡故意留有断裂——拖动你的炸药（待命区）架桥，把孤立的元件串成一条龙。
- **观赏连锁**：连锁用确定性的"激活传播"逐级炸开，连锁计数 `x N` 飙升、屏震递增、满屏烟花——爽点全在"看它炸开"。

## 连锁元件

| 元件 | 行为 |
|---|---|
| 💣 **炸药** | 激活后范围爆炸,引爆范围内所有元件与气球 |
| 🎆 **烟花** | 向四周射出火花,覆盖最广,可同时点燃多个元件 |
| 🁢 **多米诺** | 只推倒最近的一个,排好顺序是关键 |
| 🚀 **火箭** | 朝箭头方向飞出,跨越鸿沟点燃远端 |
| 🎈 **气球** | 目标,引爆即得分;相邻气球会连爆 |
| 🩷 **你的炸药** | 可任意拖放,补全断裂的链路 |

## 目标与计分

- 一次点火引爆**全部气球**即过关。
- **连锁倍率**：越往后被引爆的气球分越高（鼓励一条长龙连锁）。
- **奖励**：省下未用的炸药 +500/个、连锁长度额外加分。
- 三星评定 + 本地最佳分（localStorage 自动存档）。
- 失败（链路断裂没炸全）可**瞬间重试**——调整布局再来，是有意义的迭代而非纯运气。

## 设计概念（Game Concept）

### 核心身份

| 项 | 内容 |
|---|---|
| 核心动词 | 弹射触发 + 看连锁（**布局补全 + 一次点火**） |
| 核心幻想 | "我按下了那张多米诺骨牌的第一块"——布局家的运筹快感 |
| 独特钩子 | 像愤怒小鸟，**而且**爽点不在砸，而在"看着自己算准的连锁哗啦炸开" |
| 主导 MDA | Sensation（连锁演出） + Submission（沉浸观赏） |
| 规模 | 小到中（单文件，已含 12 关 MVP） |

### 设计支柱（Pillars）

1. **看它炸开才是高潮** —— 连锁演出本身就是奖励，节奏、声画、屏震层层递进。
   - *设计测试*：若一次连锁没有"渐强的视听爽感"，加节奏与反馈。
2. **失败是迭代,不是惩罚** —— 失败因为布局没补好（可改进），且能瞬间重试。
   - *设计测试*：若失败让人觉得"靠运气"，把不可见随机改成可见的布局问题。
3. **你来补完这台机器** —— 玩家的能动性体现在"放置缺失的一环"。
   - *设计测试*：若玩家除了点火无事可做，增加需要玩家补全的断裂点。

### 反支柱（Anti-Pillars）

- **不做刚体物理** —— 连锁用确定性激活传播，保证可预判、可设计、无失控，专注观赏性。
- **不做随机连锁** —— 同样的布局 + 起爆点永远得到同样的连锁，玩家能学习与精通。
- **不做体力 / 内购墙** —— 即点即玩，失败零成本重试。

### 视觉锚点（Visual Identity Anchor）

> **视觉铁律：连锁的每一环都要被"画"出来——一条看得见的传播之火。**

- 派对烟火美学：暗紫夜幕 + 绚烂爆炸 + 彩色气球 + 通关满屏彩纸。
- 连锁连线：每次引爆都在两元件间画一条短暂发光的"传火线"，让连锁路径可见。
- 连锁计数随长度放大、变色，最后一击慢镜定格。

### 目标玩家

- **主要**：解谜 + 观赏型玩家（喜欢"策划一场完美连锁"并欣赏它发生）。
- **次要**：成就型玩家（追求最长连锁、最少炸药的极限三星）。
- **不适合**：追求实时操作反应的玩家（本作重规划与观赏，操作仅一次点火）。

## 技术说明

- 纯 `Canvas 2D` + 原生 JS，**单文件、零依赖、零素材**（音效 WebAudio 程序化合成）。
- 连锁核心是**确定性激活事件队列**（`pending` + 延迟传播），非刚体物理 → 可预判、易设计、零失控。
- 火箭为真实飞行投射物（跨越空隙做连接），其余元件用"范围内最近 k 个"带延迟传播。
- 世界 1280×800 等比 contain 缩放，全局可见以便规划。
- 真机手感调参集中在脚本顶部「调参」区（`BOMB_R`/`FW_R`/`PROP_DELAY` 连锁节奏、`MAX_SPEED`、`SPARK_G` 等）。

## 三部曲：同源弹射的三种体验

本作与同合集的另两款同源于「愤怒的小鸟」弹射底座，构成互补三角：

| 游戏 | 体验定位 | 灵魂机制 | 主导情绪 |
|---|---|---|---|
| 🎯 弹弹变变 Sling Morph | 发射**后**操控 | 子弹时间 + 手势变身 | 操作快感 |
| 🪐 引力弹弓 Orbit Sling | 发射**前**规划轨迹 | 引力预测线 + 引力井 | 解谜发现 |
| 💥 连锁一击 Chain Bang | 一次点火 + **观赏** | 连锁激活传播 | 连锁演出 |
