# 🐑 叠了个叠（三消叠叠乐）

一个纯前端单文件的「羊了个羊」式三消叠叠乐，主打**手机即点即玩 + 晒分裂变**。

## 玩法
- 点击**没被上层压住**的方块，放进底部 7 个槽位
- 槽里凑齐 **3 个相同** 自动消除
- 7 格填满还没消除 = 游戏结束
- 道具救场：**撤销 ×3 / 洗牌 ×1 / 移出三个 ×1**
- 双关卡：第一关热身 → 第二关「地狱」（爆梗点 + 晒分）

## 运行
直接用浏览器打开 `index.html` 即可，零依赖、零构建。

```bash
open index.html        # macOS
```

## 技术要点
- 纯 `HTML/CSS/JS` 单文件，无任何外部依赖
- 方块 DOM 常驻，用 `state`（board/tray/cleared）驱动显隐，便于**撤销快照**
- 覆盖检测：被更高层、坐标重叠的方块压住则不可点
- 关卡生成保证每种花色数量为 **3 的倍数**（可解性前提）
- 自适应屏幕宽度（`computeLayout`），移动端友好
- WebAudio 简易音效，可静音
