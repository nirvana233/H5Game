# 🐭🐻🦅 变形小子 · 三态闯关

一个纯前端单文件的**马里奥式横版冒险**,核心特色是 **一键三态变身**——地形逼你随机应变,而不是靠踩蘑菇变大。

## 玩法
- 三种形态各有专属能力,**一个键循环切换**:
  - 🐭 **鼠形** — 钻进 1 格高的**窄缝**、跑得快(跳得低)
  - 🐻 **熊形** — 撞碎挡路的**脆砖墙**、碾压敌人(笨重)
  - 🦅 **鹰形** — 按住跳键**滑翔**飞越**大坑**(不能钻缝/撞砖)
- 每段地形只认对应形态,**变身循环顺序刚好对上障碍顺序**,遇障碍按一次变身键即可
- 踩怪头消灭,碰到敌人/尖刺受伤(3 颗心),掉进深坑直接重来
- 手工设计 **3 关**:关卡 1 教学 → 关卡 2 进阶组合 → 关卡 3 终点 **Boss 战**
- 通关结算用时 + 金币,记录本地最佳,可一键复制成绩晒分

## 操控
- **PC**:`← →` / `A D` 移动 · `空格` / `↑` / `W` 跳(鹰形按住滑翔)· `Shift` / `S` 变身
- **手机**:左下方向键 · 右下「跳」「变身」(松开跳键=矮跳)

## 运行
直接用浏览器打开 `index.html` 即可,零依赖、零构建。

```bash
open index.html        # macOS
```

## 技术要点
- 纯 `HTML/CSS/JS` 单文件,**Canvas 渲染**,无任何外部依赖
- **固定步长**物理(60 步/秒)+ **分轴 AABB** 瓦片碰撞,跳跃含土狼时间/输入缓冲/矮跳
- 形态决定碰撞箱尺寸:变身前检测空间是否够,不够则失败,**不会卡进墙里**
- 关卡用**语义化构建器**生成(`corridor / brittle / pit / door` 等),比手数 ASCII 更不易错
- Boss 战:击败后清除「门」砖块,终点旗激活
- 像素复古风:整数像素绘制 + 程序化 sprite + **扫描线 + 暗角**滤镜 + 视差远山
- WebAudio 8-bit 音效(跳/变身/吃币/撞碎/踩怪/受伤/过关),可静音
- 画布按视口自适应缩放,移动端虚拟按键
