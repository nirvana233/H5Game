# 🖍️ 铺路狂魔 · Path Paver

一个纯前端单文件的**画线造路反向跑酷**，主打**边跑边画的千钧一发 + 神操作晒图裂变**。全新原创玩法。

## 玩法
- 小人**自己往前冲，你控制不了他**
- **用手指在他脚下画线** → 画出平台、斜坡、桥，把坑补上
- 边跑边铺，**让他掉下去 = 结束**
- **墨水有限**（顶部进度条），画得越长耗得越多，会缓慢回充
- 沿弧线吃金币：加分 + 回墨，奖励你画漂亮的弧线
- 分数 = 铺路距离 + 金币加成

## 核心爽点
- 上帝视角"铺路救场"，千钧一发画出完美一笔的掌控快感
- 神操作和翻车都极具观赏性，墨水限制制造持续紧张

## 运行
直接用浏览器打开 `index.html` 即可，零依赖、零构建。

```bash
open index.html        # macOS
```

## 技术要点
- 纯 `HTML/CSS/Canvas/JS` 单文件，无任何外部依赖
- 实时平台物理：`surfaceAt(x)` 扫描平台 + 手绘线段，取可踩的最高表面，含台阶/斜坡攀爬容差
- 画线在世界坐标采样，按线长扣墨；离屏笔触自动剔除
- 程序化生成带间隙 + 起伏高度的平台，难度随距离提升
- 缩放系数 `S = H/720` 保证跨设备一致；奔跑摆腿、落地粒子、屏幕抖动、WebAudio 音效
