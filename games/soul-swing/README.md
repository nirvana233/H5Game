# 🌀 荡魂 · Soul Swing

一个纯前端单文件的**无跳跃荡绳跑酷**，主打**单指即玩 + 丝滑连荡爽片裂变**。全新原创玩法，不抄袭任何现有跑酷。

## 玩法
- 全程**没有跳跃键**，只有一根灵魂之绳
- **按住屏幕** → 朝最近的上方光点甩出绳子，借钟摆惯性荡起来
- **松手** → 在最佳时机脱手，把摆动甩成向前的飞跃
- 抓住下一个光点继续荡，**掉出屏幕下方 = 魂飞魄散**
- 沿弧线收集金色灵魂珠攒连击，分数 = 飞越距离 + 连击加成

## 核心爽点
- "丝滑流"零失误连荡，是天然的剪辑爽片素材
- 钟摆物理 + 轻微荡绳助力，新手也能荡两下，高手能贴边神操作

## 运行
直接用浏览器打开 `index.html` 即可，零依赖、零构建。

```bash
open index.html        # macOS
```

## 技术要点
- 纯 `HTML/CSS/Canvas/JS` 单文件，无任何外部依赖
- 钟摆约束：每帧重力积分后做绳长投影 + 径向速度消除，绳可松弛（自然手感）
- 程序化生成锚点 / 灵魂珠，难度随距离平滑提升
- 缩放系数 `S = H/720` 让物理手感跨设备一致；DPR 自适应高清
- 拖尾、粒子、视差星空、屏幕抖动、WebAudio 音效
