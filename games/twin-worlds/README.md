# 双生世界 · Twin Worlds

> 一键切换「光明 / 黑暗」两个重叠世界的横版冒险闯关。
> 光里是路的地方，暗里就是深渊——边跑边在两个现实之间精准硬切，踩着此消彼长的平台冲向终点。

纯 **Vanilla JS + Canvas 2D**，**单文件零依赖**，移动端 / 桌面端通吃。

## ▶ 运行

直接用浏览器打开 `index.html` 即可（无需构建、无需服务器、无外部资源）。

## 🎮 操作

| | 桌面 | 移动 |
|---|---|---|
| 移动 | `A` / `D` 或 `←` / `→` | 左下 `◀` `▶` |
| 跳跃 | `W` / `↑` / `空格` | 右下 `↑`（可变高度：长按跳更高） |
| **切换世界** | `Shift` / `J` | 右下 `切换` |
| 重开本关 | `R` | — |
| 玩法说明 | `H` 或点右上角 `?` | 点右上角 `?` |

> 右上角始终有一个 `?` 按钮，随时点开「玩法说明」浮层（打开时自动暂停），点任意处 / 按任意键关闭。

## 🧩 核心玩法

- **双世界同屏可见**：当前世界的平台是**实线·实体**（可踩）；另一个世界的平台是**虚线·幽灵**（踩不到）。
- **瞬时硬切**：按切换键，两个世界整屏反色互换——刚才踩着的平台可能瞬间消失，所以**切换时机就是一切**。
- **公平的死亡，慷慨的重来**：坠落 / 尖刺 / 挤压会死，但死亡 < 1 秒即秒回最近**检查点**，鼓励一次次重试。
- **看穿才能通关**：沿途收集宝石 ◆，刷新每关**最佳时间**（本地存档）。

## 📦 关卡

5 关循序渐进：初识切换 → 空中交替 → 深渊连切 → 双色登塔(垂直) → 终章·全功。

## 🛠 技术要点

- 固定步长物理（重力 / 可变高度跳跃 / coyote time / jump buffer），逐轴 AABB × 瓦片碰撞。
- 切换「挤压」公平化：优先把玩家顶出实体，顶不出才判死。
- WebAudio 程序合成全部音效（跳跃 / 切换 / 落地 / 收集 / 死亡 / 过关），零音频文件。
- 全程序化绘制（纯色块 / 几何图形 + 幽灵虚线轮廓），零图片资源。
- 响应式视野：按屏幕缩放，保证横向 ≥16 列预判空间。
- 顿帧 / 屏幕震动 / 整屏反色闪光 / 粒子等汁感反馈。

## 📐 设计支柱

1. **一眼看穿，瞬间抉择** — 状态永远清晰，失败只因"反应慢"而非"没看清"。
2. **单键魔法** — 核心只有"切换"一个键，深度全从时机与组合中生长。
3. **公平的死亡，慷慨的重来** — 死得明白、回得飞快。
4. **每关一个新念头** — 每关引入/组合一个双世界小花样。
