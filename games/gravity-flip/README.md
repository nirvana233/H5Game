# 转个向 · Gravity Flip

> 你不控制小球——你控制**重力指向哪**。哪面墙是「下」，小球就往哪掉。四面墙都能当地板走。

一款原创 2D 平台冒险 H5 小游戏。把「跳」换成了「掰重力」：按一个方向，重力立刻转向那边，小球被甩向对应的墙面并贴着滚动。

## 核心玩法

- 🔽 按上 / 下 / 左 / 右 = 重力立刻转向那一边
- 🏀 **过坑技巧**：先按侧向让小球加速，再按「下」，保留横向冲量把它甩成抛物线落到对岸
- 🧱 贴着任意一面墙都能行走 · ⚠️ 红刺碰到就重来
- 🎯 滚到金色出口星即过关 · ★ 顺路收集星星

世界不旋转、只有重力箭头和小球动，规避了「翻转晕」。右上角箭头实时指示当前重力方向。

## 操作

| | 设定重力方向 |
|---|---|
| **PC** | 方向键 / WASD |
| **手机** | 屏下十字方向盘 |

## 关卡

6 间单屏房间递进：教学（侧向重力=移动）→ 上墙翻越 → 荆棘地板绕天花板 → 中央方块绕行 → 双刺带精准翻转 → 翻墙落点终间。带最佳用时/翻转次数记录与分享。

## 技术

- 纯单文件 `index.html`，零依赖，Canvas 2D 渲染
- 向量重力 + 四向贴墙摩擦，平滑朝向旋转与速度拖影
- 运行时强制补全房间边界（安全网，防漏墙飞出）
- WebAudio 程序化音效（按方向变调）、粒子 / 震屏 juice
- 触屏十字盘 + 键盘双控，自适应缩放
- 全部 6 间经重力滑动 BFS 校验：从起点必可到达出口

直接用浏览器打开 `index.html` 即玩。
