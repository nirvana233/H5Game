# 🀄 字生万物（汉字炼成）

一个纯文字、单文件的**汉字合成发现**游戏。把两个字撞进炼炉，按真实的造字逻辑炼出新字——主打**收集成瘾 + 截图裂变 + 边玩边涨知识**。

> 木 ＋ 木 ＝ 林，林 ＋ 木 ＝ 森，日 ＋ 月 ＝ 明，女 ＋ 子 ＝ 好。

## 玩法
- 点亮两个已发现的字，放进**炼炉**自动合成
- 命中配方 → 炼成新字，点亮图鉴；凑不出 → 提示重试
- **灵感 💡**：卡住时高亮一个能合成新字的原料，给方向
- **长按**（手机）/ **右键**（桌面）任意已得字，查看它的**由来释义**
- 目标：点亮全部 **44** 个汉字，封圣成「造字之圣」
- 进度自动存档（`localStorage`），随时回来继续收集

## 运行
直接用浏览器打开 `index.html`，零依赖、零构建。

```bash
open index.html        # macOS
```

## 技术要点
- 纯 `HTML/CSS/JS` 单文件，国风宣纸 UI，移动端优先
- 配方网络数据驱动：**28 条会意配方 + 16 基础字**，键无序（`[a,b].sort()`）
- 经 Node 实测校验：无重复键、无重复产物、**28 个产物全部可达**（收集 100% 可完成、无死字）
- 释义文案内嵌每个字，长按即看「原来如此」的传播点
- `localStorage` 持久化收集进度；WebAudio 音效可静音
