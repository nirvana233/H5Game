# 🎯 弹弹变变 · Sling Morph

> 空中变身弹射 · 单文件零依赖 H5 游戏
> 直接用浏览器打开 `index.html` 即可游玩（手机 / 桌面通吃）。

---

## 一句话

愤怒的小鸟「发射后听天由命」，**弹弹变变「发射后才是正戏」**——
拉弓射出去后，**按住屏幕进入子弹时间**慢镜规划，用手势让弹丸**当场变身**，
一发打出一套连招清场。

## 核心玩法

```
拉弓蓄力(角度+力度) → 松手发射 → 飞行中按住=子弹时间 → 手势变身×N → 命中清场
                                          ↑ 本作灵魂：发射后的操控
```

每发弹丸有 **3 次变身机会**，按住屏幕减速后用手势选择招式：

| 手势 | 招式 | 效果 |
|---|---|---|
| ⬆️ 上滑 | 🪂 **滑翔** | 减重抬升、抹平弹道，越过高墙、延长射程 |
| ⬇️ 下滑 | 🪨 **俯冲** | 加重急坠，穿透木箱、击破护甲怪 |
| ⬅️➡️ 左右滑 | 🚀 **疾冲** | 横向爆发变向，急转抓墙后目标 |
| 👆 轻点 | 💥 **震爆** | 原地 AOE 震波，清掉一片敌人与易碎块 |

## 关卡机关

- 🟦 **玻璃** —— 一碰即碎
- 🟫 **木箱** —— 需俯冲穿透或多次撞击
- ⬜ **石墙** —— 不可破坏，只能绕过（滑翔/疾冲）
- 🟩 **弹床** —— 高弹力，借力飞越
- 🟥 **尖刺** —— 碰到弹丸直接报废
- 😈 **普通怪 / 🛡 护甲怪** —— 护甲怪需俯冲或震爆才能击破

## 目标与计分

- 清空全部敌人即过关；每关弹丸数量有限。
- **连击**：同一发内连续击杀，分数倍率叠加。
- **剩弹奖励**：省下的弹丸每个 +800。
- 三星评定 + 本地最佳分记录（localStorage 自动存档）。

## 设计概念（Game Concept）

### 核心身份

| 项 | 内容 |
|---|---|
| 核心动词 | 弹射 + 空中变身择时 |
| 核心幻想 | "那一发我操作得行云流水"——从瞄准的人变成空中的指挥家 |
| 独特钩子 | 像愤怒小鸟，**而且**子弹飞出去后还归你操控，深度全在"何时变、变成啥" |
| 主导 MDA | Challenge（精通） + Sensation（操作快感） |
| 规模 | 小到中（单文件，已含 12 关 MVP） |

### 设计支柱（Pillars）

1. **发射后才是正戏** —— 任何机制都要强化"飞行中的操控"，而非削弱它。
   - *设计测试*：若某关让"发射即注定"，违背支柱，砍掉或重设。
2. **一指可玩** —— 拉弓、子弹时间、变身全用一根手指，手机优先。
   - *设计测试*：若一个操作需要第二根手指或精密多键，重新设计为手势。
3. **看得见的爽** —— 顿帧、慢镜、屏震、粒子、连击飘字，每个动作都要有 juice 回馈。
   - *设计测试*：若一次击杀没有"声画双反馈"，补上再发布。

### 反支柱（Anti-Pillars）

- **不做塌楼刚体物理** —— 不引入旋转刚体堆叠，避免单文件物理出 bug、抢走"变身走位"的焦点。
- **不做体力 / 内购墙** —— 关卡即点即玩，不卡进度、不逼氪。
- **不做复杂数值养成** —— 招式固定四式，深度来自玩家手法而非数值堆叠。

### 视觉锚点（Visual Identity Anchor）

> **视觉铁律：每个动作都要有"果冻般"的弹性反应。**

- 圆润胖墩的造型 + 挤压拉伸（squash & stretch），弹丸像有弹性的小生物。
- 糖果色高对比：天蓝渐变背景、青蓝弹丸（变身时变色）、品红敌人、暖棕木箱。
- 高可读性优先——一眼能分清"哪是目标、哪是障碍、哪是危险"。
- 变身瞬间用光环 + 慢镜暗角强调"高光时刻"，天然适合截图分享。

### 目标玩家

- **主要**：休闲 + 技巧型玩家（Achiever / 喜欢复刷高分炫技）。
- **次要**：解谜型玩家（每关都有"该用哪招过"的最优解）。
- **不适合**：寻求长线养成或剧情叙事的玩家。

## 技术说明

- 纯 `Canvas 2D` + 原生 JS，**单文件、零依赖、零素材**（音效用 WebAudio 程序化合成）。
- 自研轻量物理：单圆弹丸 vs 静态 AABB 方块，子步推进防穿模。
- 世界高度固定 720，宽度随屏幕比例自适应，相机横向跟随弹丸。
- 存档：`localStorage`（关卡星级 + 最佳分）。
