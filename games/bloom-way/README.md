# 一路开花 · Bloom Way

> 没有现成的路——你走到哪，就把路种到哪。脚下没台子？随手撒颗种子，它当场「砰」地开成一朵弹力花当跳板。

一款原创 2D 平台冒险 H5 小游戏。你不是走过关卡，你是一边走一边把路**种**出来。最自由表达的玩法：同一个坑，一百种种法。

## 核心玩法

- 🌱 点击（或触摸）空中任意位置 = 种出一朵**弹力花**当跳板，消耗 1 颗种子
- 🦘 落到花上自动弹起；**触地瞬间按跳**可弹得更高（超级弹跳，连花连跳像踩节拍）
- ♻️ 踩到真正的实地，种子回满；花会**凋谢**，别磨蹭
- 🎯 抵达终点大花即过关 · ⚠️ 荆棘碰到重来
- 白色圈 = 可种范围；绿色预览 = 可种，红色 = 太远 / 被占

过深的坑要**边落边补种**第二朵花接力。

## 操作

| | 移动 | 跳 | 种花 |
|---|---|---|---|
| **PC** | ← → / A D | 空格 / ↑ | 鼠标点击空中 |
| **手机** | 左下方向键 | 右下「⤒」 | 点画面任意处 |

## 关卡

5 关递进：教学（两个坑学种花）→ 高台需弹力花 → 荆棘地弹跳跨越 → 连续坑种花节奏 → 综合终关。带最佳用时/种花数记录与分享。

## 技术

- 纯单文件 `index.html`，零依赖，Canvas 2D 渲染
- 即时生成跳板（弹力花），有限种子 + 落地回充 + 凋谢计时
- 弹跳增益窗口（触花瞬间按跳触发超级弹跳）、瞄准预览
- WebAudio 程序化音效、粒子 / 震屏 juice
- 鼠标点击 / 触摸种花，触屏 + 键盘双控，自适应缩放
- 全部关卡经间距仿真校验：无超纲段，均可经种花机制通过

直接用浏览器打开 `index.html` 即玩。
