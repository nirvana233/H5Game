# 🕵️ 字里藏凶（一段证词找真凶）

一个纯文字、单文件的**逻辑推理**游戏。一桩微型案件，几句证词，**真凶必说谎、其余皆诚实**——读懂证词，揪出唯一说得通的真凶。主打**每日打卡 + 比拼用时 + 纯逻辑硬核**。

## 玩法
- 每桩案子**有且仅有一个**逻辑自洽的真凶
- 规则：**清白者句句属实，真凶句句是假**
- 点嫌疑人选中 → 按钮指认；指认错了会**告诉你错在哪句话的矛盾**
- 两种模式：
  - **📅 每日疑案**：按日期出题，全网同题，比拼破案用时
  - **♾️ 无尽推理**：嫌疑人 3→6 递增，连破计数，刷新最佳

## 运行
直接用浏览器打开 `index.html`，零依赖、零构建。

```bash
open index.html        # macOS
```

## 技术要点
- 纯 `HTML/CSS/JS` 单文件，牛皮纸卷宗侦探 UI，移动端优先
- **程序化命题逻辑**：证词存为可在任意「凶手假设」下求值的结构（指认 / 排除 / 成对断言）
- **保证唯一解**：生成后枚举所有嫌疑人假设校验，仅当恰好一人自洽才采用
- 经 Node 实测（n=3~6 各 3000 局）：**0% 走兜底、平均 1.2~1.7 次命中、0 逻辑错误**
- 答错时自动定位**第一处逻辑矛盾**并用中文解释，兼具爽感与教育性
- `mulberry32` 种子随机源 → 每日同种子同题可复现；`localStorage` 存最佳连破
