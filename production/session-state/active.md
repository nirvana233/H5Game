# Active Prototype Session

## Concept

老板来了! 摸鱼伪装赛

## Prototype

`boss-is-coming-core-loop`

## Hypothesis

If the player holds to slack off and releases to fake work as the boss approaches, they will feel tense, greedy, and amused by near-miss escapes. We will know this is true if the player repeatedly tries to release later for a higher score after being warned.

## Path

HTML — browser prototype.

## Scope

- Test one mechanic: hold to slack off, release to fake work before inspection.
- Include one boss patrol with readable warning states.
- Include score, risk multiplier, near-miss feedback, and caught state.
- Cut menus, save systems, multiple roles, multiple task types, audio, asset production, and production architecture.
