# Game Concept: 合成宇宙 Cosmos Merge

> Status: Concept Draft
> Created: 2026-06-20
> Stage: Concept Exploration
> Codename / Dir: `games/cosmos-merge`

## The Pitch
从一粒星尘开始,同类天体相撞合成、一路演化出吞噬万物的黑洞——看你的宇宙在指尖膨胀到多大分数。

## Core Fantasy
玩家是**宇宙的造物主**:亲手把混沌尘埃一步步推演成陨石、月球、行星、恒星,直到诞生吞噬一切的黑洞。每一次合成都是一次"创世",从尘埃到黑洞的尺度跨越带来史诗般的解压快感。

## Signature Mechanics
核心三动词,3 秒即可理解:
1. **投放 (Drop)** — 左右移动选择落点,松手让当前天体落下(单指/鼠标)。
2. **合成 (Merge)** — 两个同级天体相撞融合,升级为下一级天体并得分。
3. **规划 (Plan)** — 在有限容器内安排落点与连锁,避免堆到顶部。

## The Hook
Suika / 合成大西瓜在全球验证过的"合成爽点"机制,叠加两层差异化:
- **宇宙演化主题**:从尘埃到黑洞的视觉与情感升华,自发光天体 + 合成粒子爆发,带来史诗感而非可爱风。
- **黑洞清场机制(核心创新)**:合成出的黑洞**不是终点而是清场技**——它会吞噬周围一圈天体、爆发连锁得分后消失,给玩家"绝地翻盘 / 清理死局"的策略爽点。这把"终极合成体 = 游戏结束"的常见死局,变成了一个可主动制造的爆发时刻。

## Core Loop
观察「下一个」天体 → 选落点投放 → 同类相撞合成升级(可触发连锁)→ 得分 + 三重反馈(发光/粒子/音效)→ 容器逐渐填满制造压力 → 冲击更高级天体或主动合成黑洞清场 → 容器满顶则结束 → 查看分数 / 刷新最高分 → 再来一局。

单局时长约 **3–5 分钟**,失败即重开,围绕"差一点爆顶"的紧张感制造再来一局冲动。

## Player Experience
- **Session Length**: 3–5 分钟一局,碎片化,适合移动端随手玩。
- **Difficulty Philosophy**: fair(公平的紧张)——零学习成本上手,失败永远归因于自己的布局而非随机,鼓励"我再规划得好一点"。
- **Progression**: 主线是局内技巧掌握 + 高分天梯(本地最高分);可选解锁层为天体皮肤主题(无付费墙,纯成就)。
- **Target Audience**: 全年龄休闲玩家,移动端为主,微信 / 抖音 / 小红书等截图分享场景。

## Emotional Arc
- **开局**:轻松好奇,小天体随手放,看它们滚动碰撞。
- **中期**:容器变满,开始权衡落点与连锁,紧张感上升。
- **高潮**:连锁合成或黑洞清场的爆发快感,屏幕被光与粒子点亮。
- **收尾**:爆顶的惋惜 + 刷新最高分的成就感 → 立刻想再来一局。

## Scope
**Tiny–Small**。单一核心机制(物理合成)+ 轻量自写圆形物理。纯单文件 HTML + Canvas2D,零依赖。MVP 数天内可出可玩原型;黑洞机制、皮肤主题、排行可作为 MVP 后增量。

## Inspirations & References
- **Suika Game / 合成大西瓜** — 核心合成机制与"容器爆顶"失败条件来源。
- **2048** — 合成升级的数值快感与"再合一个更高级"的牵引。
- **差异化**:宇宙演化主题(自发光视觉)+ 黑洞清场机制(终极体二次利用,化解死局)。区别于本合集已有的 `dieyao-sheep`(三消叠叠类),机制本质不同(物理碰撞合成 vs 层叠消除)。

## Open Questions
> 这些进入 /map-systems 与 /design-system 时逐一定稿。
- **黑洞机制规则**:吞噬范围(固定半径?吞噬数量上限?)、得分公式、黑洞是否触发即消失、形成条件(由中子星合成?需要两个中子星?)。
- **物理引擎**:自写圆形物理的稳定性(重叠分离、低速休眠、爆顶判定)是核心技术风险。是否引入极轻量物理还是纯手写需在引擎/原型阶段验证。
- **天体序列**:层级数量(尘埃→陨石→月球→行星→恒星→中子星→黑洞,约 7 级)是否够长,各级半径/得分曲线。
- **辅助系统**:是否提供「下一个」预览、暂存(Hold)位、撤销。MVP 至少保留「下一个」预览。
- **输入统一**:移动端触控拖放与桌面端鼠标的手感统一。
- **性能**:发光 / 粒子 / bloom 在低端移动端的帧率预算。

## Visual Identity Anchor
深邃星空黑底 + 霓虹辉光自发光天体 + 合成瞬间的粒子爆发。色彩饱和度与亮度随天体等级递增——从灰扑扑的尘埃,到璀璨发光的恒星,到吞噬一切、扭曲光线的黑洞——营造"宇宙在手中被一点点点亮"的史诗解压感。冷色为主(深蓝/紫/青),高级天体引入暖色高光(橙红恒星)形成对比。具体在 /art-bible 展开。
